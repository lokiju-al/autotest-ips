Тест-кейсы по странице задач

Подготовительная часть:
1) Открыть https://github.com/login
2) Залогиниться со своими логином и паролем
3) Открыть https://github.com/${login}/autotest-ips/issues

#001
Пользователь должен иметь возможность успешно создавать задачи с допустимым количеством символов в названии
Шаги воспроизведения:
2) Нажать кнопку Новая задача
3) Заполнить поле названия задачи с использованием сгенерированной строки
4) Нажать кнопку Создания задачи
Ожидаемый результат: Название созданной задачи соответствует введенному названию в шаге 3

#002
Пользователь не должен иметь возможность успешно создавать задачи с названием из 1025 символов
Шаги воспроизведения:
2) Нажать кнопку Новая задача
3) Заполнить поле названия задачи строкой из 1025 символов
4) Нажать кнопку Создания задачи
Ожидаемый результат: Отображается сообщение об ошибке количества символов в названии

#003
Пользователь должен иметь возможность успешно добавлять в задачу файлы допустимого формата
Шаги воспроизведения:
2) Нажать кнопку Новая задача
3) Заполнить поле названия задачи с использованием сгенерированной строки
4) Загрузить в задачу аттач-картинку допустимого формата
5) Нажать кнопку Создания задачи
Ожидаемый результат: В задаче отображается загруженный аттач-картинка

#004
Пользователь не должен иметь возможность успешно добавлять в задачу файлы недопустимого формата

3) Загрузить в задачу файл недопустимого формата

#005
Пользователь должен иметь возможность оставлять комментарии, если они включены
Шаги воспроизведения:
2) Нажать кнопку Новая задача
3) Заполнить поле названия задачи с использованием сгенерированной строки
4) Нажать кнопку Создания задачи
5) Запомнить url созданной задачи
6) Открыть меню пользователя
7) Нажать кнопку разлогинации
8) Открыть https://github.com/login
9) Залогиниться с логином и паролем комментатора
10) Открыть url созданной задачи
11) В поле комментария ввести случайно сгенерированную строку
12) Нажать кнопку сохранения комментария
Ожидаемый результат: В задаче отображается комментарий, который ввели в пункте 13

#006

Шаги воспроизведения:

3) Заполнить поле названия задачи с использованием сгенерированной строки
4) Нажать кнопку Создания задачи
5) Нажать кнопку блокировки комментариев
6) В открывшемся окне подтвердить блокировку комментариев
7) Запомнить url созданной задачи
8) Открыть меню пользователя
9) Нажать кнопку разлогинации
10) Открыть https://github.com/login
11) Залогиниться с логином и паролем комментатора
12) Открыть url созданной задачи
Ожидаемый результат: В задаче отображается сообщение, что комментарии отключены

#008
Пользователь должен иметь возможность найти задачу по существующему тегу
Шаги воспроизведения:
1) Открыть https://github.com/${login}/autotest-ips/labels
2) Нажать кнопку Новый тег
3) Заполнить поле названия тега с использованием сгенерированной строки
4) Нажать кнопку Создания тега
5) Открыть https://github.com/${login}/autotest-ips/issues
6) Нажать кнопку Новая задача
7) Заполнить поле названия задачи с использованием строки из пункта 3
8) Нажать кнопку Создания задачи
9) Нажать кнопку выбора тегов
10) Заполнить поле поиска тега строкой из пункта 3
11) В списке найденных тегов выбрать созданный тег
12) Открыть https://github.com/${login}/autotest-ips/labels
13) Заполнить поле поиска по тегу строкой из пункта 3
14) В списке найденных тегов выбрать созданный тег
Ожидаемый результат: Название найденной задачи соответствует строке из пункта 3
