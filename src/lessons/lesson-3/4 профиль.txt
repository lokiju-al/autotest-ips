#004
Пользователь должен успешно сохранить имя из одного символа через поле ввода Name

Подготовительная часть:
1) Открыть https://github.com/login
2) Ввести в поле Username or email address → LOGIN
3) Ввести в поле Password → PASSWORD
4) Нажать кнопку Sigh in
5) Открыть https://github.com/settings/profile
Шаги воспроизведения:
1) Ввести в поле Name → q
2) Нажать кнопку Update profile

Ожидаемый результат: На странице https://github.com/lokiju-test над логином lokiju-test 
отображается имя q
