Тест кейсы по странице логинации

#001
Пользователь должен успешно залогиниться в аккаунт по своему логину и паролю
Шаги воспроизведения:
1) Открыть https://github.com/login
2) Ввести в поле логинации свой логин
3) Ввести в поле ввода пароля свой пароль
4) Нажать кнопку логинации
5) Открыть меню пользователя
Ожидаемый результат: В меню пользователя отображается логин пользователя

#002
Пользователь должен успешно залогиниться в аккаунт по своей почте и паролю
Шаги воспроизведения:

3) Ввести в поле ввода пароля свой пароль
4) Нажать кнопку логинации

Ожидаемый результат: В меню пользователя отображается логин пользователя

#003
Пользователь не должен успешно залогиниться в аккаунт по неверному паролю
Шаги воспроизведения:
1) Открыть https://github.com/login
2) Ввести в поле логинации свой логин
3) Ввести в поле ввода пароля неверный пароль
4) Нажать кнопку логинации
Ожидаемый результат: Отображается сообщение об ошибке логинации
