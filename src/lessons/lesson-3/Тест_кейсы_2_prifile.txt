Тест-кейсы по странице профиля

Подготовительная часть:
1) Открыть https://github.com/login
2) Ввести в поле логинации свой логин
3) Ввести в поле ввода пароля свой пароль
4) Нажать кнопку логинации
5) Открыть https://github.com/settings/profile

#001
Пользователь должен иметь возможность изменить свое имя
Шаги воспроизведения:
1) Ввести в поле ввода имени имя, сгенерированное случайным образом
2) Нажать на кнопку сохранения
3) Открыть https://github.com/${login}
Ожидаемый результат: На странице отображается имя пользователя, которое ввели в шаге 2

#002
Пользователь должен иметь возможность изменить свою биографию
Шаги воспроизведения:
1) Ввести в поле биографии биографию, сгенерированную случайным образом
2) Нажать на кнопку сохранения
3) Открыть https://github.com/${login}
Ожидаемый результат: На странице отображается биография, которую ввели в шаге 2

#003
Пользователь должен иметь возможность изменить свое местоимение
Шаги воспроизведения:
1) Выбрать в комбобоксе выбора местоимений свое местоимение
2) Нажать на кнопку сохранения
3) Открыть https://github.com/${login}
Ожидаемый результат: На странице отображается местоимение, которое ввели в шаге 2

#004
Пользователь должен иметь возможность загрузить свой аватар
Шаги воспроизведения:
1) Загрузить первый аватар для сравнения
2) Запомнить первый аватар
3) Загрузить второй аватар, который отличается от первого
4) Запомнить второй аватар
Ожидаемый результат: В профиле отображается второй аватар

#005
Пользователь не должен иметь возможность загрузить аватар неподдерживаемого типа
Шаги воспроизведения:
1) Загрузить аватар неподдерживаемого типа
Ожидаемый результат: Отображается сообщение об ошибке типа аватара

#006
Подготовительная часть:
1) Открыть https://github.com/login
2) Ввести в поле логинации свой логин
3) Ввести в поле ввода пароля свой пароль
4) Нажать кнопку логинации
5) Открыть https://github.com/settings/emails
6) Снять галочку скрытия емейла
7) Открыть https://github.com/settings/profile

Пользователь должен иметь возможность отобразить свой емейл
Шаги воспроизведения:
1) Выбрать в комбобоксе выбора емейлов свой емейл
2) Нажать на кнопку сохранения
3) Открыть https://github.com/${login}
Ожидаемый результат: На странице отображается емейл, который ввели в шаге 4
